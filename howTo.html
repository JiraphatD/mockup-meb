How to buy e-book


เรามีความตั้งใจว่าระบบ e-book ที่ดีนั้นไม่ควรมีดีแค่อ่านสะดวกเพียงอย่างเดียว แต่จะต้องสามารถหาซื้อหนังสือมาอ่านได้ง่ายๆอีกด้วย ดังนั้นเพื่อความสะดวกแก่นักอ่านในการซื้อ e-book ทางเราได้จัดเตรียมช่องทางการสั่งซื้อและชำระเงินไว้อย่างหลากหลาย ท่านสามารถเลือกรายการ e-book ที่ต้องการ และสั่งซื้อบนอุปกรณ์ต่างๆ ทั้ง iPad / iPhone / Android / Windows / Mac ได้อย่างสะดวก โดยการสั่งซื้อ e-book มีวิธีการง่ายๆ ดังนี้จ้า

วิธีการสั่งซื้อหนังสือ (คลิกเพื่อดูรายละเอียด)

1. กดเลือกหนังสือที่ต้องการ

2. กดปุ่ม "ซื้อ"

3. กดปุ่ม "ชำระเงิน"

4. ตรวจสอบสรุปรายชื่อหนังสือที่ต้องการซื้อ แล้วกดปุ่ม "ชำระเงิน"

5. หากยังไม่ได้ล็อกอินเข้าระบบ จะต้องล็อกอินก่อน (หากล็อกอินแล้ว ให้ข้ามไปข้อถัดไป)

6. เลือกช่องทางการชำระเงินที่ต้องการ

7. กดปุ่ม "ชำระเงิน" และดำเนินการตามขั้นตอนจนชำระเงินสำเร็จ



How to pay for e-book


นักอ่านสามารถเลือกซื้อ e-book ที่ถูกใจได้ผ่านรูปแบบการชำระเงินที่หลากหลาย ไม่ว่าจะชำระผ่านบัตรเครดิต, ผ่านช่องทาง PAYSBUY , PayPal หรือหากท่านไม่มีบัตรเครดิต ก็สามารถชำระด้วยเงินสดผ่านระบบ เคาน์เตอร์เซอร์วิส 7-11 / Tesco Lotus / JustPay / MPay / Big C / Pay@Post ใกล้ๆบ้านท่าน หรือถ้าไม่สะดวกออกไปจ่าย ท่านสามารถชำระได้ด้วยการโอนเงินผ่าน ATM / Internet Banking เรียกว่าสะดวกแบบไหนจ่ายแบบนั้นกันเลยทีเดียว โดยมีขั้นตอนการชำระเงิน ดังนี้จ้า

ชำระด้วย Apple ID ผ่านอุปกรณ์ iOS (คลิกเพื่อดูรายละเอียด)

ชำระด้วยบัตรเครดิตผ่านช่องทาง Apple ID (สำหรับการสั่งซื้อบน อุปกรณ์ iOS)

  สำหรับนักอ่านที่ใช้งานแอปพลิเคชัน meb บน iPad หรือ iPhone สามารถซื้อ e-book ได้ง่ายๆ โดยชำระเงินผ่านทางระบบ In-App Purchase ของ Apple ซึ่งจะทำการตัดเงินอัตโนมัติผ่านทางบัญชีบัตรเครดิตที่ใช้ผูกกับ Apple ID รูปแบบนี้จะคล้ายกับการซื้อแอปพลิเคชันผ่านทางApple App Store คือเราเพียงแค่กรอก password ของ Apple ID เท่านั้น ระบบของ Apple จะทำการตัดเงินจากบัญชีธนาคารของ Apple ID นั้นๆ โดยที่ไม่ต้องกรอกเลขบัตรเครดิตให้ยุ่งยากเลย หลังจากนั้น e-book ที่ซื้อก็จะไปปรากฏใน My Shelf และสามารถอ่านได้ทันที และหากต้องการชำระเงินในช่องทางอื่นๆ สามารถทำได้โดยทำรายการผ่านทางเว็บไซต์ mebmarket.com เท่านั้นจ้า

  ทั้งนี้ราคาหนังสือผ่านระบบ In-App Purchase กับราคาบนช่องทางอื่นๆ อาจจะมีโอกาสแตกต่างกันได้ ขึ้นกับเจ้าของผลงานจะเป็นผู้กำหนด และอาจมีบางกรณีที่ราคาสูงกว่าช่องทางอื่นๆ ได้เพราะ

  - ราคาในระบบ In-App Purchase สามารถกำหนดเป็นขั้นๆ เช่น 35, 69, 99, ... เท่านั้น จึงไม่สามารถตั้งราคาจำหน่ายโดยอิสระ เช่น 80 บาทได้

  - ระบบ In-App Purchase นั้น ทาง Apple คิดค่าธรรมเนียมสูงกว่าช่องทางอื่นๆ ทางเจ้าของผลงานจึงสามารถกำหนดส่วนลดผ่าน In-App Purchase ได้น้อยกว่าช่องทางอื่นๆ

1. เลือก E-book เล่มที่ต้องการ กดปุ่ม “Buy $1.99”


2. จากนั้นจะมีหน้าต่างขึ้นมา ให้ใส่รหัสผ่านของ Apple ID ของอุปกรณ์นั้น แล้วกดปุ่ม “ตกลง”


3. จากนั้นจะมีหน้าต่างยืนยันการซื้อ กดปุ่ม “ซื้อ” ระบบจะดำเนินการชำระเงินโดยหักจากบัญชีธนาคารของ Apple ID

ระบบ Gift Code เป็นนวัตกรรมการซื้อ e-book รูปแบบใหม่บนระบบ meb ที่ฉีกรูปแบบการซื้อเองอ่านเองแบบเดิมๆ โดยเปิดโอกาสให้ท่านสามารถซื้อและส่ง e-book เป็นของขวัญให้เพื่อน (Buy as Gift) ได้ง่ายๆ โดยที่หลังจากสั่งซื้อสำเร็จ ท่านจะได้รับรหัส Gift Code มา และส่งต่อให้นักอ่านท่านอื่นๆ เพื่อรับ e-book ได้ฟรีผ่านแอปพลิเคชันในระบบ meb หรือบนเว็บไซต์ mebmarket.com ทันที
